Kaggle Competitions 2017




  • regression: 每一個 test example 要回傳一個 0~1 之間的值。
  • multiclass regression: 每一個 test example 要回傳多個 0~1 之間的值。
  • binary classification: 如果可以接受 0~1 之間的值，就會被算成 regression 所以暫時沒有。
  • multiclass classification: 每一個 test example 要回傳一個類別名稱。
  • labeling: 每一個 test example 可以回傳多餘一個的標籤名稱。
